Python Basics

Input

  • Make sure a variable is set to receive and store the user input
  • Use "\n" to make sure the input is only accepted in the next line.
Code
x = input("What is your name?\n")
print(f"Hello {x}")



# Output
What is your name?          #This is the input method
Lilly                       #This is the user-input response
Hello Lilly                 #This is the response printed out


String Formatting

Using an F-String

Code
aName = "Ruby"
age = 19
owing = 25
print(f"Hello {aName}, you are {age} years old. You owe ${owing:.2f}")

# Output
Hello Ruby, you are 19 years old. You owe $25.00


Using format()

Code
bName = "Cassy"
age = 30
owing = 36.45128
greeting = "Hello {}, you are {} years old. You owe ${:.2f}"
print(greeting.format(bName, age, owing))
# Output:
Hello Cassy, you are 30 years old. You owe $36.45

F String Printing

  • Allows the printing of expressions within a string.
  • Used instead of string concatenation
Code
name = "Jane"
print(f"Hello {name}")

# Output:
Hello Jane


F String Formatting

2 Decimal Places

Code
num1 = 1269
num2 = 0.57286


def multiply(a, b):
return a * b


print(f"{multiply(num1, num2):.2f}")
# Output
726.96

Conditional Statements

If ... Else

Code
a3 = ""
b3 = 5
if (a3 == a3 and b3 == 5):
print(f"a is defined, b is defined")
elif (a3 != a3 and b3 == 5):
print(f"a is not defined, b is defined")
elif (a3 == 3 and b3 != 5):
print(f"a is defined, b is not defined")
else:
print(f"a and b are not defined")
        
# Output:
a is defined, b is defined


Switch Statement - Match & Case

Code
q = input("What is your fave fruit?\n")
match q:
case "apple":
    print("You need to eat this once a day")
case "banana":
    print("This shit is bananas. B A N A N A S!")
case "kiwi":
    print("Do you know the haka?")
case "strawberry":
    print("Valentines day is calling you.")
case _:
    print("AH! Fruits are just sugar bombs anyway!")
# Output:
What is your fave fruit?                    #This was asked by the program
banana                                      #This was the user typed response
This shit is bananas. B A N A N A S!        #This was the output

    num -= 1
