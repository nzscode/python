Lists - [Square Brackets]

  • Ordered
  • Changeable : items within can be changed, added, removed
  • Indexed
  • Can have duplicates, because items are stored by index.
  • Can store String("anything within single/double quotations"), int(Integers/Floats), and Boolean(True/False) data types
Default CSS
x = [1.912, "Tim", True]
for i in range(len(x)):
    print(f"At index {i}: {x[i]}, is of type {type(x[i])}")

// At index 0: 1.912, is of type <class 'int'>
// At index 1: Tim, is of type <class 'str'>
// At index 2: True, is of type <class 'bool'>


Constructing a List

  • Must use double round brackets when creating the list with initial values
Default CSS
myList = list(("Shoes", "t-shirts", "pants"))
print(myList)

// ['Shoes', 't-shirts', 'pants']


Accessing List Items

Using the Index Numbers

Default CSS
myList = ["Shoes", "t-shirts", "pants"]
print(myList[1])

// t-shirts

Using a range of Index Numbers

Default CSS
myList = ["dogs", "cats", "rabbits", "birds", "hamsters", "ferrets"]
print(f"Length of List : {len(myList)}")
print(f"Items from index 3 to the end {myList[3:]}")
print(f"Items from the start to index 4{myList[:4]}")
print(f"Items from index 2 to the 5{myList[2:5]}")
                       // Length of List : 6
// Items from index 3 to the end ['birds', 'hamsters', 'ferrets']
// Items from the start to index 4['dogs', 'cats', 'rabbits', 'birds']
// Items from index 2 to the 5['rabbits', 'birds', 'hamsters']




Checking if an Item Exists

Default CSS
myList = ["dogs", "cats", "rabbits", "birds", "hamsters", "ferrets"]
if "rabbits" in myList:
    x = myList.index("rabbits")
    print(f"Yes, rabbits exist in this list at index {x}")

// Yes, rabbits is in myList at Index 2




Change the values of a List

Change the values of a single Index

Default CSS
myList = ["dogs", "cats", "rabbits", "birds", "hamsters", "ferrets"]
myList[3] = "mice"
print(myList)

// ['dogs', 'cats', 'rabbits', 'mice', 'hamsters', 'ferrets']


Change the values at a range of indexes

Default CSS
myList = ["dogs", "cats", "rabbits", "birds", "hamsters", "ferrets"]
myList[1:4] = ["mice", "snakes", "toads"]                       #When exact number of items to replace
print(f"Exact : {myList}")
myList[1:4] = ["mice", "snakes", "toads", "lizards", "fish"]    #When more number of items to replace
print(f"More : {myList}")
myList[1:4] = ["mice", "snakes"]                                #When less number of items to replace
print(f"Less : {myList}")

// Exact : ['dogs', 'mice', 'snakes', 'toads', 'hamsters', 'ferrets']
// More : ['dogs', 'mice', 'snakes', 'toads', 'lizards', 'fish', 'hamsters', 'ferrets']
// Less : ['dogs', 'mice', 'snakes', 'lizards', 'fish', 'hamsters', 'ferrets']




Add Items to a list

Append Items

  • Inserts items to the end of the list
Default CSS
myList = ["dogs", "cats", "rabbits", "birds", "hamsters", "ferrets"]
myList.append("Banana")
print(myList)

// ['dogs', 'cats', 'rabbits', 'birds', 'hamsters', 'ferrets', 'Banana']


Insert Items at Specified Index

Default CSS
myList = ["dogs", "cats", "rabbits", "birds", "hamsters", "ferrets"]
myList.insert(2, "Banana")
print(myList)

// ['dogs', 'cats', 'Banana', 'rabbits', 'birds', 'hamsters', 'ferrets']


Insert items from another iterable

  • Iterable = list, tuples, sets, dictionaries, etc.
Default CSS
myList = ["birds", "hamsters", "ferrets"]
newTuple = ("Apples", "Oranges")
newList = ["Red", "Pink"]
newDict = {
"Shapes": ["Square", "Circle"],
"Directions": ("North", "South")
}
myList.extend(newList)
print(myList)

myList.extend(newTuple)
print(myList)

myList.extend(newDict)
print(myList)

// ['birds', 'hamsters', 'ferrets', 'Red', 'Pink']
// ['birds', 'hamsters', 'ferrets', 'Red', 'Pink', 'Apples', 'Oranges']
// ['birds', 'hamsters', 'ferrets', 'Red', 'Pink', 'Apples', 'Oranges', 'Shapes', 'Directions']




Removing items from a List

Remove a specified item

  • If there is more than one occurrence, the remove() method only removes the first occurrence.
Default CSS
myList = ["birds", "hamsters", "ferrets"]
myList.remove("hamsters")
print(myList)

// ['birds', 'ferrets']


Remove the item at a specified index

Using pop()

  • If index is not specified, last item in the list will be removed
Default CSS
myList = ["birds", "hamsters", "ferrets"]
myList.pop(1)
print(myList)

// ['birds', 'ferrets']


Using del

  • Must specifiy an index
Default CSS
myList = ["birds", "hamsters", "ferrets"]
del myList[1]
print(myList)

// ['birds', 'ferrets']




Clearing / Emptying a list

Default CSS
myList = ["birds", "hamsters", "ferrets"]
myList.clear()
print(myList)

// []




Deleting a List

  • Will give an error.
Default CSS
myList = ["birds", "hamsters", "ferrets"]
del myList
print(myList)

// NameError: name 'myList' is not defined




Loop through a List



Iterating items of a list

Default CSS
myList = ["birds", "hamsters", "ferrets"]
for i in myList:
    print(i)

// birds
// hamsters
// ferrets


Iterating through items at index numbers

Default CSS
myList = ["birds", "hamsters", "ferrets"]
for i in range(len(myList)):
    print(myList[i])

// birds
// hamsters
// ferrets


Iterate using a While Loop

Default CSS
myList = ["birds", "hamsters", "ferrets"]
i = 0
while i < len(myList):
    print(myList[i])
    i += 1

// birds
// hamsters
// ferrets


Looping using list comprehension

Default CSS
myList = ["birds", "hamsters", "ferrets"]
[print(item) for item in myList]

// birds
// hamsters
// ferrets




Sorting Lists

Sort List Alphanumerically

  • Uppercase, comes Before lowercase
  • Default descending order
Default CSS
myList = ["onions", "broccoli", "garlic", "tomatoes", "Green chillies", "Bell pepper"]
myList.sort()
print(myList)

// ['Bell pepper', 'Green chillies', 'broccoli', 'garlic', 'onions', 'tomatoes']


Sort list, descending

Default CSS
myList = ["onions", "garlic", "tomatoes", "green chillies", "bell pepper"]
myList.sort(reverse=True)
print(myList)

// ['tomatoes', 'onions', 'green chillies', 'garlic', 'bell pepper']


Case-Insensitive Sort

Default CSS
myList = ["onions", "broccoli", "garlic", "tomatoes", "Green chillies", "Bell pepper"]
myList.sort(key=str.lower)
print(myList)

// ['Bell pepper', 'broccoli', 'garlic', 'Green chillies', 'onions', 'tomatoes']




Reverse Order

Default CSS
myList = []"onions", "broccoli", "garlic", "tomatoes", "Green chillies", "Bell pepper"]
myList.sort(reverse=True)
print(myList)

// ['tomatoes', 'onions', 'garlic', 'broccoli', 'Green chillies', 'Bell pepper']

List Comprehension

Basic List Comprehension

Default CSS
newList = [x for x in myList]
    print(newList)
    
// ['kiwi', 'cherry', 'apple', 'melon', 'cherry']

List Comprehension - Conditional

Using for

Default CSS 
myList = ["kiwi", "cherry", "apple", "melon", "cherry"]

newList = []

for x in myList:
    if "e" in x:
    newList.append(x)

print(newList)

// ['cherry', 'apple', 'melon', 'cherry']


Using List Comprehension

Default CSS 
newList = [x for x in myList if "e" in x]
    print(newList)
    
// ['cherry', 'apple', 'melon', 'cherry']


List Comprehension - Conditional !=

Using for

Default CSS 
myList = ["kiwi", "cherry", "apple", "melon", "cherry"]

newList = []

for x in myList:
    if x != "melon":
        newList.append(x)
print(newList)

// ['kiwi', 'cherry', 'apple', 'cherry']


Using List Comprehension

Default CSS 
myList = ["kiwi", "cherry", "apple", "melon", "cherry"]

newList = [x for x in myList if x != "melon"]
    print(newList)
    
//['kiwi', 'cherry', 'apple', 'cherry']


List Comprehension Range

Using for

newList = []
for i in range(10):

Using List Comprehension

Default CSS 
newList = [x for x in range(10)]

List Comprehension - Using Methods

Using for

newList = []
for i in myList:

print(newList)

// ['KIWI', 'CHERRY', 'APPLE', 'MELON', 'CHERRY']

Using for

for i in myList:

Using List Comprehension

Default CSS 
myList = ["kiwi", "cherry", "apple", "melon", "cherry"]
newList = ["fruity" for x in myList]
print(newList)
    
// ['fruity', 'fruity', 'fruity', 'fruity', 'fruity']


List Comprehension - Append to New List

Using for

Default CSS 
myList = ["kiwi", "cherry", "apple", "melon", "cherry"]  

newList2 = []
for x in myList:
    if x != "cherry":
    newList2.append(x)
else:
    newList2.append("orange")
    print(newList2)
    
// ['kiwi', 'orange', 'apple', 'melon', 'orange']


Using List Comprehension

Default CSS 
myList = ["kiwi", "cherry", "apple", "melon", "cherry"]

newList = [x if x != "cherry" else "orange" for x in myList]
    print(newList)
    
// ['kiwi', 'orange', 'apple', 'melon', 'orange']
