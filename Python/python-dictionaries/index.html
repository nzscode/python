• Dictionaries - {Curly Brackets}
    • Ordered
    • Changeable
    • Duplicates are NOT allowed
    • Contain keys and values
    
myDict = {
1: 0.253,
"two": 693,
3: "five",
"fruits": ["apples", "bananas", "oranges"],
"grades": {
    "English": 97,
    "Math": 85,
    "French": 99,
},
}

for key, value in myDict.items():
print(f"key is {key} : value is {value}")
    # Output: 
key is 1 : value is 0.253
key is two : value is 693
key is 3 : value is five
key is fruits : value is ['apples', 'bananas', 'oranges']
key is grades : value is {'English': 97, 'Math': 85, 'French': 99}
    • Deleting a dictionary
      • This will cause an error since the dictionary no longer exists. 
      del myDict
print(myDict)
      # Output:
NameError: name 'myDict' is not defined
    • Emptying a dictionary
      • myDict.clear()
print(myDict)
      # Output:
{}
    • Accessing items in a dictionary
      • Referring by key name
        • a = myDict[1]
print(f"a is {a}")

b = myDict["two"]
print(f"b is {b}")

c = myDict["fruits"][2]
print(f"c is {c}")

d = myDict["grades"]["English"]
print(f"d is {d}")
        # Output
a is 0.253
b is 693
c is oranges
d is 97
      • Using Get Keys method
        • dictKeys = myDict.keys()
print(f"Keys of the dict are: {dictKeys}")
        # Output: 
Keys of the dict are: dict_keys([1, 'two', 3, 'fruits', 'grades'])
      • Getting the Values of dictionary keys 
        • The Values of the Dictionary are dict_values([0.253, 693, 'five', ['apples', 'bananas', 'oranges'], 
                    {'English': 97, 'Math': 85, 'French': 99}])
      • Getting the key: value pairs of a dictionary 
        • dictValues = myDict.items()
print(f"The key: values of the Dictionary are {dictValues}")
        # Output:
The key: values of the Dictionary are dict_items([(1, 0.253), ('two', 693), (3, 'five'), 
        ('fruits', ['apples', 'bananas', 'oranges']), 
        ('grades', {'English': 97, 'Math': 85, 'French': 99})])
    • Adding an item to a dictionary
      • myDict["pets"] = {
"dog" : "Sammy",
"cat" : "Dean"
}
dictKeys = myDict.keys()
print(f"Keys of the dict are: {dictKeys}")
      # Output: 
Keys of the dict are: dict_keys([1, 'two', 3, 'fruits', 'grades', 'pets'])
    • Checking if a key exists
      • if "grades" in myDict:
print("This exists in myDict")
      # Output:
This key exists in myDict
    • Checking if a value exists
      • if "grades" in myDict.values():
print("This value exists in myDict")
      # Output:
This exists in myDict
    • Updating a dictionary
      • myDict.update({3: "twelve"})
print(f"The key value pair is: 3 : {myDict[3]}")
      # Output:
The key value pair is: 3 : twelve
    • Removing Items
      • Using "pop" method
        • myDict.pop(3)
print(myDict)
        # Output:
{1: 0.253, 'two': 693, 'fruits': ['apples', 'bananas', 'oranges'], 
'grades': {'English': 97, 'Math': 85, 'French': 99}}
      • Using the "del" keyword
        • Removes the item with the specified keyword
        del myDict["grades"]
print(myDict)
        # Output:
{1: 0.253, 'two': 693, 3: 'five', 'fruits': ['apples', 'bananas', 'oranges']}

    • Copying a Dictionary
      • Using the copy() method
        • newDict = myDict.copy()
print(newDict)
        # Output:
{1: 0.253, 'two': 693, 3: 'five', 'fruits': ['apples', 'bananas', 'oranges'], 
    'grades': {'English': 97, 'Math': 85, 'French': 99}}
      • Using the built in dict() function 
        • newDict = dict(myDict)
print(newDict)
        # Output :
{1: 0.253, 'two': 693, 3: 'five', 'fruits': ['apples', 'bananas', 'oranges'], 
    'grades': {'English': 97, 'Math': 85, 'French': 99}}
